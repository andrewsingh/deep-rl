# Deep RL
My implementations of popular deep reinforcement algorithms. Each algorithm is implemented in a single file for readability and ease of understanding.

## Algorithms Implemented
- [Deep Q-learning (DQN)](https://arxiv.org/pdf/1312.5602.pdf)
- [REINFORCE (with baseline)](https://people.cs.umass.edu/~barto/courses/cs687/williams92simple.pdf)
- [Deep Deterministic Policy Gradient (DDPG)](https://arxiv.org/pdf/1509.02971.pdf)
- [Twin Delayed Deep Deterministic Policy Gradient (TD3)](https://arxiv.org/pdf/1802.09477.pdf)

## Algorithms in Progress
- [Proximal Policy Optimization (PPO)](https://arxiv.org/pdf/1707.06347.pdf)
